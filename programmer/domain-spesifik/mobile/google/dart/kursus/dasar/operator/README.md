# $Operator$

> [**Dokumentasi Resmi**](https://dart.dev/language/operators "dart.dev") Segala Jenis Operator

```dart
// ^ Metode penulisan ini berlaku untuk semua tipe operator
void main() {
  print('\n' * 5);

  // $ metode penulisan manual
  var a = 1;

  a = a + a;

  print(a);

  // $ metode penulisan simpel
  var b = 1;

  b += b;

  print(b);

  print('\n' * 5);
}
```
