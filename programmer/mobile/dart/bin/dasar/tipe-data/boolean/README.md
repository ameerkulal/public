# bool

#### $Boolean$

Adalah tipe data yang digunakan untuk menyimpan nilai kebenaran. Di Dart, boolean hanya memiliki dua nilai, yaitu `true` dan `false`. Berikut untuk mendeklarasikan variabel dengan tipe data boolean

```dart
void main() {
  bool isTrue = true;
  bool isFalse = false;

  print(isFalse);
  print(isTrue);
}
```
